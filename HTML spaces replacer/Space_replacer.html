Filename fixer

Alter filenames to work
wherever you need

A click on any gray button will copy the results to your clipboard

Fill spaces with %20 to work in your browser. Example:
This is your file.html returns
This%20is%20your%20file.html

Remove "%20"s and replace with a space. Example:
This%20is%20your%20file.html returns
This is your file.html

snake_case adds an underscore betweeen word or number groups and lowercases everything. Example:
" This isYour%20%20file.html returns
this_is_your_file.html

spinal-case adds a dash betweeen word or number groups and lowercases everything. Example:
" This isYour%20%20file.html returns
this-is-your-file.html

camelCase removes any type of spacers and capitalizes the first letter of each word following the first. Example:
This _isYour%20other.file.html returns
thisIsYourOtherFile.html

Sanitize replaces most symbols with a single space. Example:
This _2.1Your%20-file's.html returns
This 2.1 Your -file's.html
